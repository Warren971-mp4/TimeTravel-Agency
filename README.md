# TimeTravel Agency - Webapp Interactive

Webapp pour une agence de voyage temporel fictive, créée avec IA générative.

## 🛠️ Stack Technique

- Frontend : React 18 (Vite), TypeScript
- Styling : Tailwind CSS
- Animations : Motion (Framer Motion)
- Routing : React Router DOM
- Icônes : Lucide React
- IA Integration : Google GenAI SDK

## ✨ Features

- Landing Page Immersive : Hero section avec animation canvas interactive (particules), compte à rebours dynamique et transitions fluides.
- Galerie de Destinations : Présentation de 3 époques exclusives (Paris 1889, Le Crétacé, Florence 1504) avec cartes interactives et vidéos d'ambiance.
- Chatbot IA : Assistant virtuel intelligent capable de répondre aux questions, donner des conseils et incarner la marque (propulsé par Gemini).
- Quiz de Recommandation : Algorithme interactif (4 questions) pour suggérer la destination idéale à l'utilisateur.
- Système de Réservation : Interface de sélection de dates et de voyageurs.

## 🤖 Outils IA & Transparence

- Génération de Code & Architecture : Google AI Studio (Gemini Models)
- Moteur du Chatbot : Google Gemini 3 Flash Preview
- Assets Visuels (Images/Vidéos) : Générés par IA (Gemini/Runway/Pika) pour les besoins de la maquette.
